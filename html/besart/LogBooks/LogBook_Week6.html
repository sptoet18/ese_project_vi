Week 6 Logbook

Overview

This week focused on CAN bus communication testing, elevator system integration, hardware assembly, and front-end web development. Work included validating CAN communication using an oscilloscope, monitoring UART communications from STM32 microcontrollers, wiring floor station hardware, and designing the Request Access webpage using CSS.

Tasks Completed

  • Measured CAN High and CAN Low voltages using an oscilloscope.
  • Captured and analyzed CAN communication waveforms.
  • Tested elevator operation and monitored CAN traffic between controllers.
  • Captured UART messages transmitted from STM32 microcontrollers.
  • Developed and styled the Request Access webpage using CSS.
  • Wired floor station switches into the elevator control system.
  • Installed and tested floor station status LEDs.
  • Verified communication and functionality after hardware integration.

Challenges

One challenge encountered was obtaining accurate oscilloscope measurements of the CAN bus signals. Proper probe placement and grounding techniques were required to successfully capture and analyze CAN High and CAN Low communication activity.

What I Learned

This week I gained additional experience using oscilloscopes to analyze CAN bus communication and UART data transmissions. I also learned how CSS can be used to improve webpage appearance through styling, layout management, hover effects, and custom button design.

Next Steps

  • Incorporate a better FSM
  • Begin Elevator Communication with website
  • Improve webpage navigation and overall user interface design.
  • Perform additional testing on floor station hardware and software integration.
