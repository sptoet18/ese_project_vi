Programming

Here are my steps to program the STM32.

  1. Change to only allow the external push buttons to request a floor.
  2. Change Shield Button 2b to transmit a CAN message to let the Supervisory Controller know that the door is open.
  3. Change Shield Button 3b to transmit a CAN message to let the Supervisory Controller know that the door is closed.
  4. Have the Shield LEDs 2b and 3b display if the door is open or closed.

Step 1 - Floor Request

My plan for this is to make the external and shield pins separate. I'm going to split them into different #define so that I can easily call them. For now the shield buttons won't do anything.

Step 2 & 3 - Announce Door is Open/Closed

My plan for this is to #define the CAN data for the open and close messages. Then I can make a flag that will be set when an open message is sent. The flag will be set when the button is pressed. It will be cleared once we detect that the flag is set and transmit the message.

Step 4 - LED Indicate Door is Open/Closed

My plan for this to just update the LEDs when the message changes.
